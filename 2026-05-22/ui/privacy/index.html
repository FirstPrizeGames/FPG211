Privacy Policy

개인정보 처리 안내

이 사이트는 로그인 없이 이용할 수 있으며 Google 로그인은 선택 사항입니다. 화면 설정과 직접 입력한 일부 값은 현재 브라우저에 저장되고, 로그인·양식·결제 정보는 연결된 외부 서비스에서 처리됩니다.

시행일 및 마지막 업데이트: 2026년 7월 17일

Processing model Local-first, external services

사이트는 비밀번호를 직접 수집하지 않습니다. 브라우저 저장소, Firebase Authentication, Google Forms와 Stripe는 각각 별도의 처리 범위를 가집니다.

저장 위치 현재 브라우저
계정 Google 로그인 선택 사항
외부 처리 Firebase · Google Forms · Stripe
삭제 사용자 직접 관리
01

브라우저에 저장되는 항목을 구분합니다.

테마, 언어, 접근성, 프로필 이름과 사진, 사이드바, 우클릭 메뉴, 사용량과 활동 기록, 포털의 구독 이메일과 주제는 localStorage에 저장됩니다. 사용량 기록은 Google 로그인 상태에서 사이트가 화면에 보이는 동안만 늘어나며 Firebase 계정으로 동기화되지 않습니다. 오프라인 표시를 위해 Cache Storage가 사용될 수 있습니다.

  • 기본 동작 페이지 스크립트와 오프라인 캐시는 사이트 기능을 위한 기술 항목이며 광고 프로필을 만들지 않습니다. 로그인 상태는 Firebase Authentication이 별도로 관리합니다.
  • 안내 확인 쿠키 안내에서 확인을 누르면 profile_storage_consent 쿠키가 SameSite=Lax로 최대 1년간 저장됩니다. Settings의 쿠키 설정에서 끌 수 있습니다.
  • 광고 및 분석 현재 광고 쿠키와 분석 쿠키를 사용하지 않습니다.
02

Google 로그인은 선택 사항입니다.

로그인하지 않아도 공개 페이지를 이용할 수 있으며 사용 한도는 소모되지 않습니다. Google 로그인을 선택하면 Firebase Authentication이 사용자 식별자, 이름, 이메일 주소, 프로필 사진, 로그인 제공자와 인증 상태를 처리합니다. Google 비밀번호는 이 사이트가 보거나 저장하지 않습니다. 로그인하지 않은 상태의 로컬 프로필과 구독 설정은 현재 브라우저에만 저장됩니다.

03

외부 서비스로 넘어가는 순간

일반 피드백, 기능 요청, 질문 및 Team 예약은 Google Forms로 연결될 수 있습니다. 양식에 입력한 내용은 Google의 설정에 따라 저장·처리되고 사이트 운영자가 응답을 검토할 수 있습니다. 외부 공유 링크에는 해당 플랫폼의 정책이 적용됩니다.

04

결제 정보는 Stripe에서 처리합니다.

Pro 결제 버튼은 Stripe Checkout으로 연결될 수 있으며 Team은 Google Forms 예약 방식입니다. 카드 번호, 보안 코드와 결제 인증 정보는 이 사이트가 직접 저장하지 않습니다. 최종 통화와 금액은 Stripe 화면에서 확인해야 합니다. Stripe Privacy Policy

05

언제든지 지울 수 있습니다.

Settings의 보안 항목에서 사이트 데이터를 삭제하면 localStorage, 안내 확인 쿠키와 오프라인 캐시가 지워집니다. Google 로그아웃은 현재 브라우저의 로그인 상태를 종료하지만 Firebase Authentication 계정 자체를 삭제하지는 않습니다. 계정 삭제 요청은 Feedback으로 보낼 수 있으며, 알림 권한과 Google Forms·Stripe 데이터는 각 서비스에서 별도로 관리해야 합니다.

06

알림 권한은 브라우저가 관리합니다.

알림을 허용하거나 거부한 상태는 브라우저와 운영체제가 관리합니다. 현재 이 사이트는 푸시 구독 서버나 기기 토큰 데이터베이스를 운영하지 않으며, 권한 상태를 사이트 서버로 전송하지 않습니다.

07

문의와 변경 사항을 공개합니다.

개인정보 관련 질문, Firebase Authentication 계정 삭제 요청 또는 문구 수정 요청은 Feedback 페이지에서 보낼 수 있습니다. 처리 방식이 달라지면 이 페이지의 업데이트 날짜와 관련 설명을 함께 변경합니다. Feedback 열기
