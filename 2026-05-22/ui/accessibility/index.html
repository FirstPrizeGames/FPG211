Accessibility

접근성 지원 안내

현재 사이트에 실제로 적용된 키보드 이동, 언어 선택, 화면 표시 보조, 모바일 지원 범위와 제한 사항을 안내합니다.

마지막 업데이트: 2026년 7월 18일

Reference WCAG 2.2 AA 참고

독립적인 접근성 인증이나 전체 적합성 감사를 완료했다는 의미는 아닙니다.

01

키보드와 초점

각 문서에는 본문으로 바로 이동하는 링크가 있으며, 주요 링크와 버튼에는 키보드 초점 표시를 적용합니다. 설정 탭과 날짜별 접기·펼치기는 표준 버튼을 사용합니다.

02

언어와 문서 구조

브라우저 언어에 따라 한국어 또는 영어를 자동 선택하고, 설정에서 직접 변경할 수 있습니다. 페이지 언어와 제목, 버튼 이름도 선택한 언어에 맞춰 갱신합니다.

03

화면 표시와 움직임

운영체제의 움직임 감소 설정이 켜지면 페이지 로더의 애니메이션을 줄입니다. 설정의 화면 표시 보조를 켜면 글자와 버튼이 커지고 일부 텍스트 대비가 높아지며 일부 효과가 짧아집니다.

04

모바일과 반응형 화면

모바일 메뉴와 하단 빠른 액션을 제공하며, 주요 버튼은 반응형 너비와 최소 높이를 사용합니다. 화면별 점검은 계속 진행되므로 특정 기기에서 문제가 생길 수 있습니다.

계속되는 점검

새 기능과 브라우저 변경으로 문제가 다시 생길 수 있어 실제 사용 환경을 계속 확인합니다.
