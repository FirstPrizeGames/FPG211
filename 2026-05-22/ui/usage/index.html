사용 한도 작동 방식

이 한도는 서버 계정이 아니라 현재 브라우저에서만 계산됩니다. 처음 접속한 순간부터 5시간 한도와 7일 주간 한도가 시작되고, 기간이 지나면 자동으로 새 한도로 초기화됩니다.

브라우저 기준 기록은 이 기기의 브라우저 저장소에만 남습니다.
자동 초기화 5시간 또는 7일 창이 지나면 다음 접속 시 새 창이 시작됩니다.
서버 저장 없음 로그인 계정이나 결제 기록과 연결되지 않습니다.
