Privacy Policy

개인정보와 사이트 설정을 투명하게 다룹니다.

이 페이지는 사이트가 어떤 정보를 사용하고, 어떤 정보는 저장하지 않는지 간단히 설명합니다.

Storage Local only
Accounts Not required
Payments External links
01

브라우저에 저장되는 설정

테마, 언어, 강조 컬러, 화면 밀도, 사이드바 여부 같은 사용자 설정은 브라우저의 localStorage에 저장됩니다.

02

계정 없이 볼 수 있는 사이트

현재 사이트는 자체 로그인, 회원가입, 서버 데이터베이스를 사용하지 않습니다.

03

외부 서비스로 이동하는 기능

Feedback, Patreon 결제, 외부 공유 기능은 해당 외부 서비스의 개인정보 처리 기준을 따릅니다.

04

설정 삭제

설정 페이지의 브라우저 캐시 정리 기능이나 브라우저 설정을 통해 저장된 사이트 데이터를 삭제할 수 있습니다.
