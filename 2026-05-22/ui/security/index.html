Security

보안 경계를 분명하게 설명합니다.

이 사이트가 브라우저에 저장하는 것, 외부 서비스로 넘기는 것, 사용자가 직접 제어하는 것을 나눠서 보여줍니다.

Security model Local-first controls

민감한 결제 정보는 사이트가 직접 저장하지 않고, 알림과 저장소 권한은 브라우저 권한을 따릅니다.

Browser storage

테마, 언어, 활동 기록, 구독 입력값은 이 브라우저의 로컬 저장소에 머뭅니다.

Cookies

필수 동작과 저장 안내 확인용 쿠키만 사용하며 광고/분석 쿠키는 현재 사용하지 않습니다.

External checkout

결제 버튼은 Stripe가 호스팅하는 페이지로 이동하며 카드 번호와 인증 정보는 이 사이트가 직접 보관하지 않습니다.

Notifications

사이트 알림은 사용자가 브라우저 권한 팝업에서 허용한 경우에만 표시됩니다.

Feedback forms

피드백 양식은 외부 서비스 정책을 따르므로 민감한 개인정보는 입력하지 않는 것이 좋습니다.

Report an issue

보안이나 개인정보 관련 문제가 보이면 Feedback 경로로 알려주세요.
