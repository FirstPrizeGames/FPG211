Accessibility Standard

접근성 기준

First Prize Games는 키보드 이동, 명확한 초점 표시, 읽기 대비, 언어 선택, 모바일 표시 기준을 핵심 접근성 기준으로 관리합니다.

마지막 업데이트: 2026년 6월 30일

Target standard WCAG 2.2 AA oriented

이 페이지는 공식 인증서가 아니라 사이트가 목표로 삼는 접근성 기준과 현재 지원 범위를 설명합니다.

Keyboard Skip link and visible focus
Contrast Readable dark UI
Language Auto, Korean, English
Controls Settings modal
Perceivable Supported

텍스트 대비, 이미지 대체 처리, 페이지 제목, 명확한 콘텐츠 구조를 우선 기준으로 둡니다.

Operable Improving

키보드 이동, skip link, 포커스 표시, 모달 닫기와 메뉴 버튼 접근성을 지속적으로 점검합니다.

Understandable Supported

한국어/영어 전환, 자동 언어 감지, 일관된 버튼 이름, 간결한 안내 문구를 유지합니다.

Robust Improving

정적 HTML 구조, 의미 있는 링크, 표준 버튼, ARIA 라벨을 사용해 다양한 브라우저에서 동작하도록 관리합니다.

01

키보드와 초점 기준

주요 페이지에는 본문으로 바로 이동하는 skip link가 있고, 주요 링크와 버튼에는 focus-visible 표시가 적용됩니다. 설정 모달, 공유, 인쇄, 우클릭 메뉴는 버튼 기반으로 열고 닫을 수 있게 관리합니다.

02

색 대비와 읽기 기준

기본 화면은 어두운 고대비 구성을 사용하며, 텍스트와 컨트롤은 공통 색상 토큰으로 관리합니다. 중요한 안내, 버튼, 선택 상태는 배경과 구분되도록 밝기와 굵기를 조정합니다.

03

언어와 이해 가능성 기준

기본 언어는 브라우저 언어를 감지해 자동으로 선택할 수 있고, 사용자는 한국어와 영어 중 원하는 언어를 설정할 수 있습니다. 페이지 제목, 버튼, 도움말 문구는 같은 의미가 유지되도록 번역합니다.

04

시각 효과 기준

사이트는 과한 블러와 애니메이션을 기본 디자인에서 제한합니다. 기기의 reduced motion 설정을 존중하도록 무거운 전환 효과를 줄이는 방향으로 관리합니다.

05

모바일 접근성 기준

모바일에서는 하단 빠른 액션, 메뉴 버튼, 설정 모달, 주요 CTA가 손가락으로 누르기 쉬운 크기를 유지하도록 점검합니다. 텍스트가 버튼 밖으로 넘치거나 서로 겹치지 않도록 반응형 폭을 제한합니다.

06

검토와 문제 신고

접근성 문제는 일반 피드백보다 우선적으로 검토합니다. 키보드 이동, 읽기 대비, 언어 전환, 모바일 표시에서 문제가 보이면 사용한 브라우저와 페이지 이름을 함께 알려주세요. Support로 문제 알리기

접근성 설정을 조정할 수 있습니다.

테마, 언어, 접근성 안내, 보안, 프로필, 우클릭 메뉴 설정은 설정 모달에서 관리합니다.
