Our principles

Build clear and trustworthy screens.

Four principles keep visible features and their descriptions aligned with how the site actually works.

  1. 01
    Clarity

    Give every page a clear job and show essential information first.

  2. 02
    Accessibility

    Review keyboard use, mobile layouts, contrast, and motion settings together.

  3. 03
    Privacy

    Describe browser storage, external sign-in, and payment boundaries as they are.

  4. 04
    Reliability

    Test visible controls and routes, then report their current status accurately.

How the site works.

The site separates information stored in the browser from actions handled by external services.

  • Browser storage Theme, language, usage, and interface preferences remain in the current browser.
  • Optional Google sign-in Google provides the account picker; this site does not ask for a separate password.
  • External services Feedback and checkout open in Google Forms and Stripe.
